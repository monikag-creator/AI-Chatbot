#  Study Assistant Chatbot
A simple keyword-based chatbot built with Python that helps students with study tips, exam preparation, time management, and motivation — all from the terminal.

## Features
- **Greeting recognition** — Responds naturally to greetings
- **Study tips** — Proven techniques like the Pomodoro method and active recall
- **Exam preparation** — Practical advice to prepare effectively before exams
- **Time management** — Tips to plan and organize your study schedule
- **Motivation** — Encouraging responses when you're feeling low
- **Help menu** — Lists all available topics anytime
- **Graceful exit** — Ends the chat with a friendly farewell

## Requirements
- Python 3.x
No external libraries needed — runs with pure Python!

##  Getting Started
1. **Clone or download** this repository.
2. **Run the script:**
```bash
python AI-chatbot.py
```

##  Usage
Once started, you'll see:
```
=======================================================
  WELCOME TO STUDY ASSISTANT CHATBOT
=======================================================
  I'm here to help you study smarter and stay motivated!
  Type 'bye!' or 'exit' to end the conversation.
=======================================================
```

Then just type your question or topic:
```
You: how to study
Bot: Here are some effective Study Tips:
  1. Break your study sessions into 25-min chunks (Pomodoro Technique).
  ...
You: I'm feeling tired
Bot: Stay Motivated:
  1. Remember your WHY — why did you start?
  ...
```

### Supported Topics

| What you type               | What you get             |
|-----------------------------|--------------------------|
| `hello`, `hi`, `hey`        | Greeting response        |
| `study`, `study tips`       | Study techniques         |
| `exam`, `test`, `revision`  | Exam preparation tips    |
| `time`, `schedule`, `plan`  | Time management advice   |
| `motivation`, `tired`       | Motivational response    |
| `help`, `options`           | List of all topics       |

### Exiting the Chat
Type any of the following to exit:
```
bye!  /  exit!  /  quit  /  goodbye!
```
